# Award Seat Alerts - Frontend

React app for award seat search and alert management.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Features

- 🔍 Search award availability across multiple airlines
- 🔔 Set up SMS alerts for your favorite routes
- 📱 Responsive design
- 🌍 Global airport support

## Build

```bash
npm run build
npm run preview
```
