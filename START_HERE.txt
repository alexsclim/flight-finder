🔄 HOW IT WORKS:

  1. USER SEARCHES:
     • Picks airports, dates, cabin class, airlines
     • Backend calls all selected airline adapters in parallel
     • Results cached for 30 minutes
     • Frontend displays findings

  2. USER CREATES ALERT:
     • Alert stored with search criteria
     • Background job checks every hour
     • Finds new matches
     • Sends SMS notification (with cooldown)
     • User can view all matches on dashboard

  3. SMS NOTIFICATIONS:
     • Twilio sends alerts to user's phone
     • Respects user's cooldown preference
     • Example: "✈️ Award found! JFK→LHR on Jun 5 Business • 62.5k miles • United"
